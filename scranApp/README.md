# Meal Prep MVP

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints.

To test on your phone, connect phone and laptop to the same Wi-Fi and open the network URL Vite prints, e.g.

```text
http://192.168.x.x:5173
```

## Current scope

- Recipe bank
- Pick meals
- Adjust servings
- Pantry / already-have toggles
- Combined shopping list
- Copy shopping list
- Cook mode preview
